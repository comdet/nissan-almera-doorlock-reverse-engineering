# Car Companion — State Machine

## รถ: Nissan Almera 1.0 Turbo (N18) เกียร์ CVT (P R N D L)

## CAN Bus ที่ OBD-II Port

OBD port ของรถรุ่นนี้ต่อผ่าน **CAN Gateway** — เป็น diagnostic-only bus ไม่มี broadcast traffic
ข้อมูลทั้งหมดต้อง **poll (ถาม)** ทุกครั้ง ไม่สามารถฟังเฉยๆ ได้

| แหล่งข้อมูล | วิธีอ่าน | ต้นทุน |
|---|---|---|
| OBD PIDs (RPM, Speed, Throttle...) | ส่ง request ไป 0x7DF | ไม่ต้องเปิด session — ปลอดภัย |
| BCM 0x0109 (ประตู ล็อค ไฟ เบรค) | ส่ง request ไป 0x745 | ต้องเปิด ExtSession — แต่ฟรีถ้า DRL เปิดอยู่ |
| Gear 0x1301 (เกียร์ P/R/N/D/L) | ส่ง request ไป 0x7E1 | ต้องเปิด ExtSession — single frame เร็ว |
| Handbrake 0x0E07 (เบรคมือ) | ส่ง request ไป 0x743 | ต้องเปิด ExtSession — multiframe |

## 8 States

### 1. บิดกุญแจ (ACC_ON)

**สถานการณ์:** คนขับเข้ารถ บิดกุญแจไปตำแหน่ง ACC หรือ ON เครื่องยังไม่ติด

**ทำอะไร:**
- Poll RPM ทุก 2 วินาที — รอจนเครื่องติด (RPM > 0)
- อ่าน Battery — แสดงแรงดันไฟ

**ไม่ทำ:** อ่านประตู เกียร์ ไฟ (ยังไม่จำเป็น)

**หมายเหตุ:** บางครั้งคนขับข้ามขั้นนี้ไป start เลย — ไม่มีปัญหา RPM จะ > 0 ทันที กระโดดไป state ถัดไป

**ออกจาก state นี้เมื่อ:** RPM > 0 → ไปที่ "เครื่องติดแล้ว"

---

### 2. เครื่องติดแล้ว (ENGINE_ON)

**สถานการณ์:** สตาร์ทเครื่องแล้ว รถจอดนิ่ง คนขับกำลังจะออกตัว

**ทำอะไร:**
- **เปิด DRL อัตโนมัติ**
- อ่าน **เกียร์** ครั้งแรก (แสดงบน HUD ว่าอยู่ P)
- อ่าน **เบรคมือ** ครั้งแรก
- Poll RPM + Speed ทุก 1 วินาที
- Poll BCM ทุก 3 วินาที (ไฟ เบรค ล็อค)
- Poll Coolant + Battery ทุก 5 วินาที

**เตือน:** เบรคมือยังดึงอยู่!

**ออกจาก state นี้เมื่อ:**
- Speed > 0 → ไปที่ "กำลังขับ"
- RPM = 0 + เกียร์ P → ไปที่ "เพิ่งดับเครื่อง"

---

### 3. กำลังขับ ยังไม่ล็อค (DRIVING)

**สถานการณ์:** รถเริ่มเคลื่อนที่ ขับในซอย ยังไม่ถึงความเร็วที่จะล็อค (< 20 km/h)

**ทำอะไร:**
- Poll HUD ทุก 500ms — RPM, Speed, Throttle, คันเร่ง
- Poll BCM ทุก 2 วินาที — ไฟเลี้ยว, ไฟหน้า, ไฟสูง, เบรค
- Poll Coolant + Battery ทุก 3 วินาที

**HUD แสดง:** RPM, Speed, Throttle, ไฟเลี้ยว, ไฟหน้า, เบรค

**ออกจาก state นี้เมื่อ:**
- Speed >= 20 km/h → **สั่ง LOCK** → ไปที่ "ล็อคแล้ว กำลังวิ่ง"
- RPM = 0 + เกียร์ P → ไปที่ "เพิ่งดับเครื่อง"

---

### 4. ล็อคแล้ว กำลังวิ่ง (LOCKED_CRUISING)

**สถานการณ์:** ขับบนถนนใหญ่ ความเร็ว >= 20 km/h ประตูล็อคแล้ว

**ทำอะไร:**
- Poll HUD ทุก 500ms — RPM, Speed, Throttle, คันเร่ง
- Poll BCM ทุก 2 วินาที — ไฟเลี้ยว, ไฟหน้า, ไฟสูง, เบรค (แสดง HUD)
- Poll Coolant + Battery ทุก 3 วินาที
- Poll MIL + Catalyst + Ambient ทุก 15 วินาที

**เตือน:**
- เครื่องร้อนเกิน! (Coolant > 100°C)
- แบตต่ำ! (< 12V)
- ขับเร็วเกิน! (> 120 km/h)
- รอบสูงเกิน! (> 6000 RPM)
- Check Engine! (MIL)
- แคทร้อนเกิน! (> 800°C)

**HUD แสดงตลอด:** RPM, Speed, Throttle, คันเร่ง, ไฟเลี้ยว, ไฟหน้า, ไฟสูง, เบรค, Coolant, Battery, Ambient

**ออกจาก state นี้เมื่อ:**
- Speed = 0 → ไปที่ "ล็อคแล้ว รถจอด"
- RPM = 0 + เกียร์ P → ไปที่ "เพิ่งดับเครื่อง"

---

### 5. ล็อคแล้ว รถจอด (LOCKED_STOPPED)

**สถานการณ์:** รถจอดนิ่ง เครื่องยังติด ประตูยังล็อค — ติดไฟแดง, รถติด, จอดเติมน้ำมัน, จอดเยี่ยว

**ทำอะไร:**
- Poll RPM + Speed ทุก 1 วินาที
- Poll BCM ทุก 2 วินาที — **เฝ้าดูประตูเปิด-ปิด**
- Poll เกียร์ ทุก 3 วินาที — แยก idle stop (เกียร์ D, RPM=0) กับดับเครื่องจริง (เกียร์ P, RPM=0)

**Idle Stop:** ถ้ารถมีระบบ idle stop (ดับเครื่องเองตอนจอด) RPM จะ = 0 แต่เกียร์ยังอยู่ D → **ไม่ปลดล็อค** อยู่ state เดิม ปล่อยเบรคเครื่องติดใหม่ขับต่อได้เลย

**Scenario ย่อย:**

| สถานการณ์ | เกิดอะไร | ทำอะไร |
|---|---|---|
| ติดไฟแดง | จอดนิ่ง ไม่เปิดประตู | ไม่ทำอะไร อยู่ state เดิม |
| รถติด | จอดๆ วิ่งๆ | speed > 0 → กลับ "ล็อคแล้ว กำลังวิ่ง" (ไม่ต้องล็อคใหม่ ยังล็อคอยู่) |
| จอดเติมน้ำมัน | คนขับเปิดประตูลง ปิดประตู | detect door event → ไปที่ "รอล็อคใหม่" |
| จอดเยี่ยว | ผู้โดยสารเปิดประตูลง ปิดประตู | detect door event → ไปที่ "รอล็อคใหม่" |
| Idle stop ทำงาน | RPM=0 แต่เกียร์ D | ไม่ทำอะไร อยู่ state เดิม |

**ออกจาก state นี้เมื่อ:**
- Speed > 0 → กลับ "ล็อคแล้ว กำลังวิ่ง"
- ประตูเปิดแล้วปิด → ไปที่ "รอล็อคใหม่"
- RPM = 0 + เกียร์ P → ไปที่ "เพิ่งดับเครื่อง"

---

### 6. รอล็อคใหม่ (REARM)

**สถานการณ์:** มีคนเปิดประตูแล้วปิด (ลงไปเติมน้ำมัน, เยี่ยว, ส่งผู้โดยสาร) รอให้รถวิ่งถึง 20 km/h จะล็อคประตูอีกรอบ — **Circular Locking**

**ทำอะไร:**
- Poll RPM + Speed ทุก 1 วินาที
- Poll BCM ทุก 2 วินาที
- Poll เกียร์ ทุก 3 วินาที

**ออกจาก state นี้เมื่อ:**
- Speed >= 20 km/h → **สั่ง LOCK อีกรอบ** → กลับ "ล็อคแล้ว กำลังวิ่ง"
- RPM = 0 + เกียร์ P → ไปที่ "เพิ่งดับเครื่อง"

---

### 7. เพิ่งดับเครื่อง (ENGINE_OFF)

**สถานการณ์:** คนขับดับเครื่อง (RPM = 0 + เกียร์ P) ระบบนับถอยหลัง 3 วินาทีก่อนปลดล็อค

**เงื่อนไขเข้า state นี้:** RPM = 0 **และ** เกียร์ = P เท่านั้น — ถ้า RPM = 0 แต่เกียร์ D/N (idle stop) จะไม่เข้า state นี้

**ทำอะไร:**
- Poll RPM ทุก 1 วินาที — ตรวจว่าดับจริงหรือจะ restart
- นับถอยหลัง 3 วินาที

**Scenario:**

| สถานการณ์ | เกิดอะไร | ทำอะไร |
|---|---|---|
| ดับเครื่องจริง | RPM = 0 ตลอด 3 วินาที | ปลดล็อค + ปิด DRL → ไปที่ "จอดเรียบร้อย" |
| ดับแล้วเปลี่ยนใจ | RPM กลับมา > 0 ภายใน 3 วินาที | ยกเลิกทุกอย่าง → กลับ "เครื่องติดแล้ว" |

**ออกจาก state นี้เมื่อ:**
- 3 วินาทีผ่าน → **สั่ง UNLOCK + ปิด DRL** → ไปที่ "จอดเรียบร้อย"
- RPM > 0 (restart) → ยกเลิก → กลับ "เครื่องติดแล้ว"

---

### 8. จอดเรียบร้อย (PARKED)

**สถานการณ์:** ดับเครื่องแล้ว ปลดล็อคแล้ว ปิด DRL แล้ว คนขับกำลังจะออกจากรถ

**ทำอะไร:**
- อ่าน **เกียร์** — ใส่ P ยัง?
- อ่าน **เบรคมือ** — ดึงยัง?
- Poll BCM ทุก 5 วินาที — ดูประตูเปิด/ปิด
- Poll เกียร์ ทุก 10 วินาที
- Poll เบรคมือ ทุก 10 วินาที

**เตือน:**
- ไม่ได้ใส่เกียร์ P!
- ไม่ได้ดึงเบรคมือ!

**Scenario:**

| สถานการณ์ | เกิดอะไร |
|---|---|
| คนขับเปิดประตูลง | HUD แสดงว่าประตูเปิด |
| ประตูปิดหมด | HUD แสดงว่าประตูปิดครบ |
| ACC OFF | ESP32 หมดไฟ จบ |
| สตาร์ทใหม่ | RPM > 0 → กลับ "เครื่องติดแล้ว" + เปิด DRL |

## แผนภาพ State Machine

```
    ┌──────────┐
    │ บิดกุญแจ  │ poll: RPM
    │ ACC_ON   │
    └────┬─────┘
         │ RPM > 0
         ▼
    ┌───────────┐
    │ เครื่องติด  │ poll: RPM Speed BCM เกียร์ เบรคมือ
    │ ENGINE_ON │ → เปิด DRL
    └─┬───────┬─┘
      │       │ speed > 0
      │       ▼
      │  ┌──────────┐
      │  │ กำลังขับ   │ poll: RPM Speed Throttle BCM
      │  │ DRIVING   │
      │  └─────┬────┘
      │        │ speed >= 20
      │        ▼
      │  ┌────────────────┐
      │  │ ล็อคแล้ว วิ่งอยู่ │ → สั่ง LOCK
      │  │ LOCKED_CRUISING│ poll: HUD ทั้งหมด + BCM
      │  └──┬──────┬──────┘
      │     │      │ speed = 0
      │     │      ▼
      │     │ ┌────────────────┐
      │     │ │ ล็อคแล้ว จอดอยู่ │ poll: RPM Speed BCM(ประตู) เกียร์
      │     │ │ LOCKED_STOPPED │ ← ไฟแดง / รถติด / จอดปั๊ม
      │     │ └─┬────────┬────┘
      │     │   │        │ ประตูเปิด-ปิด
      │     │   │        ▼
      │     │   │  ┌───────────┐
      │     │   │  │ รอล็อคใหม่  │ poll: RPM Speed BCM เกียร์
      │     │   │  │ REARM     │ ← Circular Locking
      │     │   │  └─────┬────┘
      │     │   │        │ speed >= 20 → สั่ง LOCK
      │     │   │        │ ──→ กลับ LOCKED_CRUISING
      │     │   │        │
      │     │   │  RPM=0 + เกียร์ P (ทุก state ข้างบน)
      │     │   │        │
      ▼     ▼   ▼        ▼
    ┌──────────────┐
    │ เพิ่งดับเครื่อง │ poll: RPM (ยืนยันดับจริง)
    │ ENGINE_OFF   │ นับ 3 วินาที
    └──────┬───────┘
           │ 3 วินาทีผ่าน
           ▼
    ┌────────────┐
    │ จอดเรียบร้อย │ → สั่ง UNLOCK + ปิด DRL
    │ PARKED     │ poll: BCM(ประตู) เกียร์ เบรคมือ
    └────────────┘ เตือน: ไม่ได้ใส่ P / ไม่ได้ดึงเบรคมือ
```

## Idle Stop (ระบบดับเครื่องอัตโนมัติ)

Nissan Almera มีระบบ idle stop — จอดนิ่งเครื่องดับเอง ปล่อยเบรคเครื่องติดเอง

**ปัญหา:** RPM = 0 เหมือนดับเครื่องจริง ถ้าไม่แยก จะปลดล็อคตอนจอดไฟแดง

**วิธีแยก:**
- Idle stop: RPM = 0 แต่เกียร์ **D หรือ N** (ยังอยู่ตำแหน่งขับ)
- ดับเครื่องจริง: RPM = 0 + เกียร์ **P** (ต้องใส่ P ถึงจะดับได้)

**กฎ:** เข้า ENGINE_OFF ได้ก็ต่อเมื่อ RPM = 0 **และ** เกียร์ = P เท่านั้น
