## แผนการ Decode ถัดไป

ต้องเทียบค่าโดยเปลี่ยนสถานะทีละอย่าง:

1. **เปิดประตูคนขับ** → อ่าน DID 0x0109 เทียบ (หา door open flag)
2. **เปิดประตูผู้โดยสาร** → อ่านอีกรอบ (หา door บานไหน)
3. **เปลี่ยนเกียร์ P→R→N→D** → อ่าน DID 0x0108 เทียบ (หา gear byte)
4. **ดึงเบรคมือ/ปล่อย** → อ่าน DID 0x0108 เทียบ (หา handbrake)
5. **เปิดไฟหน้า** → อ่าน DID 0x0E07 เทียบ (หา headlight flag)
6. **เปิดไฟเลี้ยว** → อ่าน DID 0x0E07 เทียบ (หา turn signal)
7. **ดับเครื่อง** → อ่าน DID 0x1301, 0x1304 เทียบ (หา engine off value)
